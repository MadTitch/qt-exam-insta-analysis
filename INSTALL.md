# 설치 가이드 — 시험 분석 콘텐츠 생성 스킬 (카드뉴스 + A4 분석지)

방금 끝난 학교 시험을 분석해 ① **인스타 카드뉴스**(1080×1080 카드 세트) 또는 ② **A4 상세 분석지**(인쇄용 PDF)를 만들어 주는 Claude Code 스킬입니다. 실행하면 어떤 형태로 만들지 먼저 물어봅니다. 시험지 문항은 자동으로 잘라 넣고 정답·채점 표기는 가려서 넣습니다.

## 설치
운영체제에 맞는 파일을 실행하세요. (스킬을 `~/.claude/skills/` 에 복사합니다.)

- **Windows**: `install.bat` 더블클릭
- **Mac / Linux**: 터미널에서
  ```
  cd <이 폴더>
  bash install.sh
  ```

## 준비물 (첫 실행 시 자동 설치)
- **Python 3** (Mac/Linux 기본 내장, Windows는 python.org 설치 — "Add to PATH" 체크)
- 렌더링에 **playwright(chromium) · Pillow · numpy** 를 씁니다 — 스킬이 첫 실행에서 자동 설치합니다. (인터넷 필요)
- 한글 폰트 **Noto Sans CJK KR** (대부분 환경에 이미 있음)

> 이 스킬은 **로컬에서 완결**됩니다. 별도 구독 서버·API 키가 필요 없습니다.

## 쓰는 법
Claude Code에서 한국어로:
1. 시험지 사진(여러 장) 또는 PDF, 그리고 시험범위 텍스트를 올린다.
   **교과서 본문·어휘표·정답** 등 원본 자료가 있으면 함께 — "교과서를 어떻게 비틀었는지" 변형 분석까지 됩니다.
2. 원하는 것을 말한다:
   - "시험 분석 카드뉴스 만들어줘" → 📱 인스타 모드
   - "상세 분석지 만들어줘" → 📄 A4 리포트 모드
   - "시험지 분석해줘" → 어떤 형태로 만들지 물어봅니다 (둘 다 선택 가능)
3. **전 문항 분석표**를 보여주면 심층분석할 문항을 고른다.
4. (카드뉴스) **디자인 샘플**에서 색 팔레트 + 테마를 고른다. / (분석지) 악센트 색만 정한다.
5. 완성 — 카드 PNG 세트 또는 A4 PDF가 생성됩니다.

## 디자인 옵션 (카드뉴스)
- **색 팔레트 10종**: sunset / navy / ocean / teal / forest / plum / berry / rose / gold / slate
- **레이아웃 테마 9종**: A 블루프린트 · B 미니멀 · C 볼드매거진 · D 다크네온 · E 파스텔 · F 그라디언트 · G 뉴스페이퍼 · H 네온사인 · I 손그림
- 색과 테마는 독립 조합. 학원명·강사명·색은 매번 입력받습니다.

## 저작권 안내
시험지를 통째로 게시하지 않고, 문항을 캡처한 뒤 **정답·채점 표기 등 일부를 가려** 활용합니다(교육청 안내 기준). 분석에 필요한 최소 부분만 잘라 넣으며, 학생 이름·점수 등 개인정보는 배제합니다.
