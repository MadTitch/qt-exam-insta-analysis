# 설치 가이드 — 시험지 분석 인스타 카드뉴스 스킬

방금 끝난 학교 시험을 분석해 **학원 홍보용 인스타그램 캐러셀(1080×1080 카드 여러 장)** 을 만들어 주는 Claude Code 스킬입니다. 시험지 문항을 자동으로 잘라 넣고 정답·채점 표기는 가려서 넣습니다.

## 설치
운영체제에 맞는 파일을 실행하세요. (스킬을 `~/.claude/skills/` 에 복사합니다.)

- **Windows**: `install.bat` 더블클릭
- **Mac / Linux**: 터미널에서
  ```
  cd <이 폴더>
  bash install.sh
  ```

## 준비물 (첫 실행 시 자동 설치)
- **Python 3** (Mac/Linux 기본 내장, Windows는 python.org 설치)
- 카드 렌더링에 **playwright(chromium) · Pillow · numpy** 를 씁니다 — 스킬이 첫 실행에서 자동으로 설치합니다. (인터넷 필요)
- 한글 폰트 **Noto Sans CJK KR** (대부분 환경에 이미 있음)

> 이 스킬은 **로컬에서 완결**됩니다. 별도 구독 서버·API 키가 필요 없습니다.

## 쓰는 법
Claude Code에서 한국어로:
1. 시험지 사진(여러 장)과 시험범위 텍스트(교재·과·문법 토픽)를 올린다. (교과서 본문·어휘표·정답 등 **원본 자료**가 있으면 함께 주면 분석이 훨씬 정확해집니다.)
2. "시험 분석 인스타 카드뉴스 만들어줘" 라고 한다.
3. 스킬이 **전체 문항 분석표**를 보여주면 심층분석할 문항을 고른다.
4. **디자인(색 팔레트 + 테마)** 을 샘플에서 고른다.
5. 표지 → 시험범위 → 배점표 → 출제특징 → (교과서 변형분석) → 심층분석 → 대비전략 → 마무리 순의 카드 PNG가 생성됩니다.

## 디자인 옵션
- **색 팔레트 10종**: sunset / navy / ocean / teal / forest / plum / berry / rose / gold / slate
- **레이아웃 테마 9종**: A 블루프린트 · B 미니멀 · C 볼드매거진 · D 다크네온 · E 파스텔 · F 그라디언트 · G 뉴스페이퍼 · H 네온사인 · I 손그림
- 색과 테마는 독립이라 자유롭게 조합됩니다. 학원명·강사명·색은 매번 입력받습니다.

## 저작권 안내
시험지를 통째로 게시하지 않고, 문항을 캡처한 뒤 **정답·채점 표기 등 일부를 가려** 활용합니다(교육청 안내 기준). 분석에 필요한 최소 부분만 잘라 넣습니다.
