# qt-exam-insta-analysis

방금 끝난 학교 시험을 분석해 **두 가지 콘텐츠** 중 원하는 것을 만들어 주는 Claude Code 스킬입니다. 실행하면 어떤 형태로 만들지 먼저 물어봅니다.

| 모드 | 산출물 |
|--|--|
| 📱 **인스타 카드뉴스** | 1080×1080 카드 여러 장 — 표지·배점표·출제특징·문항 심층분석·대비전략 |
| 📄 **A4 상세 분석지** | 인쇄용 PDF — 총평·**문항 전수 분석표**·교과서 변형 분석·고난도 상세 해설·대비전략 |

시험지 문항을 자동으로 잘라 넣고, 정답·채점 표기는 가려서 넣습니다.

## 무엇이 좋은가
- **전 문항 전수 분석** 후 한눈에 보여주고, 카드/해설에 넣을 문항만 고르면 됩니다.
- 교과서 원문을 함께 주면 **"원문을 어떻게 비틀어 함정을 만들었는지"(변형 분석)** 까지 짚습니다.
- 카드뉴스는 **색 팔레트 10종 × 레이아웃 테마 9종** 조합에서 샘플을 보고 고릅니다.
- 채점된 시험지도 OK — 빨간펜·연필 표기를 색 기반으로 자동 제거합니다.
- **로컬 완결형** — 별도 구독 서버·API 키 불필요.

## 설치
- **Mac/Linux**: `bash install.sh`
- **Windows**: `install.bat` 더블클릭

→ `~/.claude/skills/qt-exam-insta-analysis/` 에 설치됩니다. 자세한 내용은 [INSTALL.md](INSTALL.md).

## 업데이트
새 버전이 나오면 **최신 ZIP을 다시 받아 설치 파일을 한 번 더 실행**하면 됩니다 — 설치 스크립트가 기존 버전을 깨끗이 교체합니다.
- ZIP 다시 받기: 이 페이지 초록색 **`< > Code` → Download ZIP**
- git이 익숙하면: `git clone` 해두고 `git pull` 후 `bash install.sh` (Windows는 `install.bat`)
- 진행 중이던 작업물(deck.json·report.json·크롭·결과물)은 사용자 폴더에 있으므로 업데이트해도 사라지지 않습니다.

> **[초보자 가이드](초보자-가이드.md)** 를 보세요.

## 준비물
Python 3, 그리고 렌더링용 playwright(chromium)·Pillow·numpy (첫 실행 시 자동 설치), 한글 폰트 Noto Sans CJK KR.

## 쓰는 법
Claude Code에서 시험지 사진(또는 PDF) + 시험범위 텍스트(교재·과·문법·정답)를 올리고:
- **"시험 분석 카드뉴스 만들어줘"** → 인스타 모드
- **"상세 분석지 만들어줘"** → A4 리포트 모드
- 그냥 **"시험지 분석해줘"** → 어떤 형태로 만들지 물어봅니다 (둘 다도 가능)

전 문항 분석표에서 심층분석 문항을 고르고, (카드뉴스는) 디자인 샘플에서 색+테마를 고르면 완성됩니다.

## 저작권
시험지를 통째로 게시하지 않고, 문항을 캡처한 뒤 정답·채점 표기 등 일부를 가려 활용합니다. 분석에 필요한 최소 부분만 잘라 넣으며, 학생 이름·점수 등 개인정보는 배제합니다.
