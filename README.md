# qt-exam-insta-analysis

방금 끝난 학교 시험을 분석해 **학원 홍보용 인스타그램 캐러셀(1080×1080 카드 여러 장)** 을 만들어 주는 Claude Code 스킬입니다. 시험지 문항을 자동으로 잘라 넣고, 정답·채점 표기는 가려서 넣습니다.

## 무엇을 만드나
표지 → 시험범위(핵심 문법) → 유형별 배점표 → 영역별 출제특징 → 교과서 원문 대비 **변형 분석** → 문항 **심층분석**(문항 크롭) → 학교 맞춤 대비전략 → 마무리 순의 카드 묶음.

- **색 팔레트 10종** × **레이아웃 테마 9종** = 90가지 디자인 조합 (색·테마 독립)
- 심층분석 크롭은 가로형으로 시인성 확보, 정답/채점 표기는 색 기반으로 자동 제거
- 학원명·강사명·색은 매번 입력받는 재사용형
- **로컬 완결형** — 별도 구독 서버·API 키 불필요

## 설치
- **Mac/Linux**: `bash install.sh`
- **Windows**: `install.bat` 더블클릭

→ `~/.claude/skills/qt-exam-insta-analysis/` 에 설치됩니다. 자세한 내용은 [INSTALL.md](INSTALL.md).

## 준비물
Python 3, 그리고 렌더링용 playwright(chromium)·Pillow·numpy (첫 실행 시 자동 설치), 한글 폰트 Noto Sans CJK KR.

## 쓰는 법
Claude Code에서 시험지 사진 + 시험범위 텍스트(교재·과·문법·정답)를 올리고 **"시험 분석 인스타 카드뉴스 만들어줘"** 라고 하면 실행됩니다. 전체 문항 분석표에서 심층분석 문항을 고르고, 디자인(색+테마) 샘플에서 선택하면 카드 PNG가 생성됩니다.

## 저작권
시험지를 통째로 게시하지 않고, 문항을 캡처한 뒤 정답·채점 표기 등 일부를 가려 활용합니다. 분석에 필요한 최소 부분만 잘라 넣습니다.
