# 디자인 스펙 — 단일 원천 (레퍼런스 게시물과 톤 일치)

CSS 구현은 `assets/base.css`(공통) + `assets/theme_a|b|c.css`(테마). 이 문서는 "왜/얼마"의 기준이다.

## 캔버스
- 1080 × 1080 px. 렌더 시 `device_scale_factor=2` (실제 2160² 산출).
- 배경: 메인색 + 옅은 blueprint grid(흰색 10% 라인, 54px 격자).

## 색 (메인색에서 파생 — 하드코딩 금지)
build_cards.py 의 `derive_palette()` 가 `brand.color` 에서 계산해 `:root` 에 주입한다.

| 변수 | 의미 | 파생 규칙 |
|---|---|---|
| `--brand`      | 메인색(배경)         | 입력 그대로 |
| `--brand-dark` | 제목/강조            | 메인색 × 0.70 (어둡게) |
| `--brand-ink`  | 가장 진한 텍스트     | 메인색 × 0.35 |
| `--brand-tint` | 헤더/행 음영         | 메인색 + 흰색 82% 혼합 |
| `--brand-tint2`| 더 옅은 톤           | 메인색 + 흰색 92% 혼합 |

기본 메인색: 네이비블루 `#3B4FD8`. 사용자가 다른 색을 주면 그 색으로 전부 재파생된다.

## 폰트
- `Noto Sans CJK KR` (환경 기본 설치 가정). 제목 900(Black), 본문 500/700.
- 영어가 섞여도 동일 폰트로 통일.

## 레이아웃 공통
- 폴더 탭: 상단 좌측, 섹션 라벨. 흰 배경 + 굵은 검정 외곽선, 아래 모서리만 본문 카드와 연결.
- 본문: 흰 라운드 카드(border-radius 28px) + 6px 검정 외곽선.
- 하단 우측: 책더미 일러스트(SVG, card.css 의 `.books`).
- 하단 중앙: 워터마크 = 학원명.
- 표지: 큰 제목/부제 중앙, 강사명 칩(선택), 포인트 일러스트.

## 표(배점표)
- 헤더 = `--brand-dark` 박스 + 흰 글자.
- 행 교차 음영(zebra).
- 숫자(문항번호·배점)는 우측 정렬, 배점은 `--brand-dark` 굵게 강조.

## 텍스트 넘침 방지
- feature_list/strategy 본문은 카드 높이를 넘으면 잘린다. 항목 4~5개, 본문 2~3줄 권장.
- 렌더 후 반드시 `view` 로 잘림 점검 → 길면 문구를 줄이거나 항목 수를 줄인다(폰트 축소는 최후수단).
