# 🐣 왕초보 가이드 — Claude Code 앱 설치부터 시험 분석 콘텐츠 만들기까지

> 컴퓨터·프로그래밍을 하나도 몰라도 따라 할 수 있게 아주 잘게 적었습니다.
> **터미널(검은 명령어 창) 없이, 앱(GUI)** 으로 진행합니다. 위에서부터 순서대로 하세요.
> 막히면 맨 아래 [문제 해결](#8-잘-안-될-때-문제-해결)을 보세요.

---

## 0. 이 스킬이 하는 일 (30초 요약)

방금 끝난 학교 영어 시험지 **사진(또는 PDF)** 과 **시험범위**를 주면, 두 가지 중 원하는 것을 만들어 줍니다:

- 📱 **인스타 카드뉴스** — 학원 홍보용 정사각형 이미지 여러 장 (표지→배점표→출제특징→심층분석→대비전략)
- 📄 **A4 상세 분석지** — 인쇄해서 나눠줄 수 있는 PDF (전 문항 분석표·교과서 변형 분석·고난도 해설)

실행하면 **어떤 형태로 만들지 먼저 물어봅니다.** (둘 다 만들 수도 있어요)
쓰려면 **Claude Code 앱**이 필요합니다. 아래에서 하나씩 준비할게요.

---

## 1. 준비물 (딱 3가지)

1. **Claude 유료 구독** — Claude Code는 **Pro 또는 Max**(팀/기업 플랜 포함) 구독이 **필수**입니다. [claude.ai](https://claude.ai) 에서 가입·결제해 두세요. *(API 키는 필요 없음)*
2. **인터넷**
3. **시간 10~15분** — 처음 한 번만 세팅하면 됩니다.

---

## 2. Claude Code 앱 설치

### 2-1. 앱 다운로드
쓰는 컴퓨터에 맞는 링크를 클릭해 설치 파일을 받으세요.

- **Mac** (인텔·애플실리콘 공통): <br>https://claude.ai/api/desktop/darwin/universal/dmg/latest/redirect
- **Windows** (대부분 이것, 64비트): <br>https://claude.ai/api/desktop/win32/x64/setup/latest/redirect
- **Windows** (ARM 노트북, 예: Surface Pro X): <br>https://claude.ai/api/desktop/win32/arm64/setup/latest/redirect
- **Linux** (베타): 안내 → https://code.claude.com/docs/en/desktop-linux

### 2-2. 설치
- **Mac**: 받은 `.dmg` 파일 더블클릭 → Claude 아이콘을 **Applications(응용 프로그램) 폴더로 드래그** → 응용 프로그램에서 실행.
- **Windows**: 받은 `.exe` 파일 더블클릭 → 안내대로 **다음/설치** 누르기.

### 2-3. ⚠️ Windows 사용자만 — "Git" 먼저 설치
Windows에서는 **Git for Windows** 가 있어야 앱이 정상 작동합니다.
- https://git-scm.com/download/win 에서 다운로드 → 실행 → **전부 기본값(Next)** 으로 설치.
- 설치 후 Claude Code 앱을 **껐다 켜세요.**

> (Mac은 Git이 대부분 기본 포함이라 신경 안 써도 됩니다.)

## 3. 앱 로그인 (한 번만)

1. 설치한 **Claude Code 앱 실행**
2. 로그인 화면이 나오면 **Anthropic(claude.ai) 계정으로 로그인** → 브라우저가 열리면 승인
3. 한 번 로그인하면 저장돼서 다음부터는 자동입니다.

## 4. 카드 그림 도구 준비 — Python (한 번만)

카드 이미지를 그리는 데 **Python**이 필요합니다.

- **Mac**: 대부분 이미 있습니다. 없으면 앱이 안내하는 대로 설치하거나 [python.org](https://www.python.org/downloads/) 에서 설치.
- **Windows**: [python.org/downloads](https://www.python.org/downloads/) 에서 설치. 설치 첫 화면 맨 아래 **"Add Python to PATH"** 를 **꼭 체크**하고 설치하세요.

> 카드 그리는 세부 도구(playwright·Pillow·numpy)는 스킬을 **처음 실행할 때 자동으로 설치**됩니다. 신경 안 쓰셔도 돼요.

---

## 5. 이 스킬 설치하기

### 5-1. 스킬 파일 내려받기
1. 이 페이지( https://github.com/MadTitch/qt-exam-insta-analysis ) 위쪽 초록색 **`< > Code`** 버튼 → **Download ZIP**
2. 받은 ZIP을 **압축 해제**(더블클릭). `qt-exam-insta-analysis-main` 폴더가 생깁니다.

### 5-2. 설치 (가장 쉬운 방법)
- **Windows**: 압축 푼 폴더 안의 **`install.bat`** 를 **더블클릭**. "설치 완료" 창이 뜨면 끝!
- **Mac**: 아래 **"Claude에게 부탁하기"** 방법이 제일 쉽습니다. 👇

### 5-3. Mac에서 가장 쉬운 방법 — Claude에게 시키기
터미널 명령이 부담되면, **앱에서 Claude에게 설치를 시키면 됩니다.**
1. Claude Code 앱을 열고 (4단계처럼) 아무 폴더나 **Select folder** 로 연다.
2. 압축 푼 `qt-exam-insta-analysis-main` 폴더를 채팅 입력창에 **드래그**해서 넣는다.
3. 이렇게 입력:
   ```
   이 폴더 안의 qt-exam-insta-analysis 스킬을 ~/.claude/skills 에 복사해서 설치해줘
   ```
   Claude가 알아서 복사해 줍니다. (권한 물어보면 **허용/Yes**)

> 설치 위치는 `~/.claude/skills/qt-exam-insta-analysis/` 입니다. **앱과 (혹시 쓰는) CLI가 이 폴더를 함께 봅니다.**

---

## 6. 앱에서 작업 폴더 열기

1. 앱 상단의 **"Code"** 탭 선택
2. 첫 메시지 전에 아래를 고릅니다:
   - **Environment: Local** (내 컴퓨터에서 실행 — 꼭 이걸로)
   - **Select folder**: 결과물을 저장할 폴더 선택 (예: 바탕화면에 만든 `시험분석` 폴더)
   - **Model**: 기본값 그대로
   - **Permission mode**: 초보자는 **"Ask permissions"**(물어보고 실행) 추천
3. 이제 아래 입력창에 한국어로 말하면 됩니다.

---

## 7. 실제로 만들어 보기

1. 시험지를 **사진으로** 찍거나 **PDF**로 준비해 컴퓨터에 저장합니다. (여러 장이면 다 준비)
2. 시험범위를 **글**로 준비합니다. (교재·과·문법 토픽. 교과서 본문·어휘표·정답까지 주면 "교과서를 어떻게 비틀었는지" 변형 분석까지 됩니다)
3. 앱 입력창 옆 **"+" 버튼 → Attach files** 로 시험지를 첨부(또는 창에 **드래그앤드롭**). 시험범위 글은 붙여넣기.
4. 원하는 것을 입력:
   ```
   시험 분석 카드뉴스 만들어줘        ← 인스타용
   상세 분석지 만들어줘               ← A4 인쇄용 PDF
   시험지 분석해줘                    ← 뭘 만들지 물어봐 줌
   ```
   또는 입력창에 **`/`** 를 치면 설치된 스킬 목록이 뜹니다 → **`/qt-exam-insta-analysis`** 선택.
5. 그다음은 Claude가 **하나씩 물어봅니다.** 편하게 답하면 돼요:
   - (맨 처음) 카드뉴스로 만들지, A4 분석지로 만들지
   - 전체 문항 분석표 → **깊게 다룰 문항** 고르기
   - 카드뉴스면 **색 + 디자인 테마** 샘플에서 고르기 / 분석지면 포인트 색만
   - 학원명·강사명·표지 문구 입력
6. 완성되면 결과물(카드 이미지들 또는 report.pdf)이 **선택한 폴더**에 저장되고, Claude가 위치를 알려줍니다. 카드는 인스타에 순서대로 올리고, PDF는 그대로 인쇄하면 끝! 🎉

## 8. 잘 안 될 때 (문제 해결)

| 증상 | 해결 |
|--|--|
| 앱이 로그인/구독 오류 | [claude.ai](https://claude.ai) 에서 Pro/Max 구독·결제 상태 확인 |
| (Windows) 앱 기능이 이상하거나 제한됨 | **Git for Windows**(2-3) 설치 후 앱 재시작 |
| `/` 눌러도 스킬이 안 보임 | 5단계 설치가 안 된 것. 설치 위치 `~/.claude/skills/qt-exam-insta-analysis/` 확인. 앱 재시작. |
| 카드 만들 때 python/playwright 오류 | Python(4단계) 설치 확인 + 인터넷 연결. 첫 실행은 도구 설치로 시간이 좀 걸림. |
| 한글이 네모(□)로 깨짐 | 한글 폰트(Noto Sans CJK KR)가 없음. "노토 산스 CJK KR 설치" 검색해 설치 |
| 결과 폴더를 못 찾음 | 6단계에서 **Select folder** 로 고른 폴더를 확인. Claude에게 "결과 어디 저장했어?" 라고 물어보기 |

그래도 막히면 앱에서 **증상을 한국어로 그대로 말하면** Claude가 도와줍니다.
예: "카드 만들다가 이런 오류가 났어: (오류 내용 붙여넣기)"
