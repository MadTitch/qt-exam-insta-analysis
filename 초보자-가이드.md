# 🐣 왕초보 가이드 — Claude Code 설치부터 시험 분석 카드 만들기까지

> 컴퓨터·프로그래밍을 하나도 몰라도 따라 할 수 있게 아주 잘게 적었습니다.
> **위에서부터 순서대로** 하나씩 하시면 됩니다. 중간에 막히면 맨 아래 [문제 해결](#7-잘-안-될-때-문제-해결)을 보세요.

---

## 0. 이 스킬이 하는 일 (30초 요약)

방금 끝난 학교 영어 시험지 **사진**과 **시험범위**를 주면, 학원 인스타에 올릴 수 있는
**분석 카드뉴스(정사각형 이미지 여러 장)** 를 자동으로 만들어 줍니다.
표지 → 시험범위 → 배점표 → 출제특징 → 교과서 변형분석 → 문항 심층분석 → 대비전략 → 마무리.

이걸 쓰려면 **"Claude Code"** 라는 프로그램이 필요합니다. 아래에서 하나씩 깔아 볼게요.

---

## 1. 미리 알아둘 3가지 (준비물)

1. **Claude 계정 + 유료 플랜** — Claude Code는 Anthropic의 유료 구독(Claude Pro 또는 Max 등)이나 API 크레딧으로 돌아갑니다. [claude.ai](https://claude.ai) 에서 가입하고 결제해 두세요.
2. **인터넷** — 설치와 실행에 필요합니다.
3. **시간 10~20분** — 처음 한 번만 세팅하면 됩니다.

> 💡 용어가 낯설어도 괜찮아요. 아래 명령어는 **그대로 복사해서 붙여넣기** 하면 됩니다.

---

## 2. "터미널" 여는 법 (명령어 입력하는 검은 창)

프로그램 설치는 **터미널**(명령어 입력 창)에서 합니다.

- **Mac**: `⌘(커맨드) + 스페이스` → `터미널` 이라고 입력 → 엔터. 검은(또는 흰) 창이 뜹니다.
- **Windows**: 시작 버튼 → `PowerShell` 검색 → **Windows PowerShell** 클릭.

이 창에 명령어를 붙여넣고 **엔터**를 치면 실행됩니다. (붙여넣기: Mac `⌘+V`, Windows `Ctrl+V` 또는 마우스 오른쪽 클릭)

---

## 3. Node.js 설치 (Claude Code를 돌리는 엔진)

Claude Code는 **Node.js** 위에서 돌아갑니다. 먼저 이것부터 깝니다.

1. [https://nodejs.org](https://nodejs.org) 접속
2. **LTS** 라고 적힌 큰 버튼 클릭 → 설치 파일 다운로드
3. 다운로드한 파일 실행 → **계속/다음** 만 누르면 설치 완료 (Windows는 "Add to PATH" 체크가 기본으로 되어 있음 — 그대로 두기)

설치가 됐는지 확인 — 터미널에 아래를 붙여넣고 엔터:
```
node -v
```
`v20.x.x` 처럼 숫자가 나오면 성공입니다. (숫자 아무거나 나오면 OK)

---

## 4. Claude Code 설치

터미널에 아래를 붙여넣고 엔터:
```
npm install -g @anthropic-ai/claude-code
```
- 설치에 1~3분 걸립니다. 글자가 주르륵 올라가다 멈추면 끝난 거예요.
- (Mac에서 `permission denied` 가 나오면 → 맨 아래 [문제 해결](#7-잘-안-될-때-문제-해결) 참고)

> 설치 방법은 업데이트될 수 있어요. 잘 안 되면 **docs.anthropic.com** 에서 "Claude Code" 를 검색해 최신 안내를 보세요.

---

## 5. Claude Code 처음 실행 + 로그인

1. **작업할 폴더**로 이동합니다. 예를 들어 바탕화면에 `내작업` 폴더를 만들었다면:
   - Mac: `cd ~/Desktop/내작업`
   - Windows: `cd $HOME\Desktop\내작업`
   (폴더가 없으면 그냥 `cd ~/Desktop` 처럼 아무 폴더로 가도 됩니다.)

2. 실행:
   ```
   claude
   ```
3. 처음 실행하면 **로그인** 하라고 나옵니다. 안내대로 브라우저가 열리면 Claude 계정으로 로그인 → 승인. 터미널로 돌아오면 준비 끝!

4. 이제 `>` 표시 뒤에 한국어로 말을 걸 수 있어요. 시험 삼아 `안녕` 이라고 쳐보세요. 답이 오면 성공입니다.

> 나가고 싶을 땐 `/exit` 입력 또는 `Ctrl + C` 를 두 번.

---

## 6. 이 스킬 설치하기

### 6-1. Python 있는지 확인 (카드 이미지를 그리는 데 필요)
터미널(Claude Code 밖, 3번에서 쓴 그 창)에 붙여넣기:
```
python3 --version
```
- 숫자가 나오면 OK.
- **Windows**에서 아무것도 안 나오거나 오류면 → [python.org/downloads](https://www.python.org/downloads/) 에서 설치.
  설치 화면 맨 아래 **"Add Python to PATH"** 를 꼭 체크하세요.

### 6-2. 스킬 파일 내려받기
1. 이 페이지( https://github.com/MadTitch/qt-exam-insta-analysis ) 위쪽의 초록색 **`< > Code`** 버튼 클릭
2. **Download ZIP** 클릭 → 다운로드
3. 받은 ZIP을 **압축 해제**(더블클릭). `qt-exam-insta-analysis-main` 같은 폴더가 생깁니다.

### 6-3. 설치 실행
- **Windows**: 압축 푼 폴더 안의 **`install.bat`** 를 **더블클릭**. 창이 뜨고 "설치 완료" 가 나오면 끝.
- **Mac**: 터미널에서 그 폴더로 이동 후 설치. (폴더 경로가 헷갈리면 `cd ` 를 친 뒤 **폴더를 터미널 창으로 드래그**하면 경로가 자동 입력됩니다.)
  ```
  cd ~/Downloads/qt-exam-insta-analysis-main
  bash install.sh
  ```
  `✅ 스킬 설치 완료` 가 뜨면 끝입니다.

> 이 스킬은 **별도 구독 서버나 키가 필요 없습니다.** 첫 실행 때 카드 그림 그리는 도구(playwright·Pillow·numpy)를 자동으로 깝니다.

---

## 7. 실제로 카드 만들기

1. 시험지를 **사진으로** 찍어 컴퓨터에 저장해 둡니다. (여러 장이면 다 준비)
2. 시험범위를 **글(텍스트)** 로 준비합니다. (교재·과·문법 토픽. 교과서 본문·어휘표·정답까지 있으면 분석이 훨씬 정확해져요.)
3. 작업 폴더에서 `claude` 실행 → 시험지 사진과 시험범위를 올리고 이렇게 말하세요:
   ```
   시험 분석 인스타 카드뉴스 만들어줘
   ```
   > 사진을 올리는 법: 파일을 터미널 창으로 **드래그** 하거나 경로를 붙여넣으면 됩니다.

4. 그다음은 Claude가 **하나씩 물어봅니다.** 편하게 답하면 돼요:
   - 전체 문항 분석표를 보여주면 → **카드로 깊게 다룰 문항** 고르기
   - **색 + 디자인 테마** 샘플을 보여주면 → 마음에 드는 것 고르기
   - 학원명·강사명·표지 문구 입력
5. 다 고르면 카드 이미지들이 만들어지고, **어디에 저장됐는지 Claude가 알려줍니다.** 그 폴더에서 이미지를 꺼내 인스타에 순서대로 올리면 끝!

## 8. 잘 안 될 때 (문제 해결)

| 증상 | 해결 |
|--|--|
| `command not found: claude` | Claude Code 설치(4번)가 안 된 것. 터미널 껐다 켜고 다시 `npm install -g @anthropic-ai/claude-code`. |
| `command not found: node` / `npm` | Node.js(3번)부터 설치. 설치 후 터미널 껐다 켜기. |
| Mac에서 `npm install` 시 `permission denied` / `EACCES` | 앞에 `sudo ` 를 붙여 `sudo npm install -g @anthropic-ai/claude-code` (비밀번호 입력). |
| `python3` 를 못 찾음 (Windows) | python.org에서 설치하고 **Add to PATH** 체크. 터미널 껐다 켜기. |
| 카드 만들 때 "playwright/chromium" 관련 오류 | 인터넷 연결 확인 후 다시 시도. 첫 실행에 시간이 좀 걸립니다. |
| 한글이 네모(□)로 깨져 보임 | 한글 폰트(Noto Sans CJK KR)가 없는 것. "노토 산스 CJK KR 설치" 로 검색해 설치. |
| 로그인/구독 오류 | [claude.ai](https://claude.ai) 결제·구독 상태 확인. |

그래도 막히면, Claude Code 창에서 **한국어로 증상을 그대로 말하면** Claude가 도와줍니다. 예: "카드 만들다가 이런 오류가 났어: (오류 내용 붙여넣기)"
